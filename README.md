# Local Development

You need to install [Node](https://nodejs.org/en) for this command to work. Run:

```bash
npx http-server -c-1 -p 3000
```

- `http-server` deploys static/dynamic sites to localhost at a given port.
- `-c-1` Disables source/file caching.
- `-p 3000` Sets the server to deploy at port 3000.
